Detailed Description

Layer is a basic unit of Scene.

Each layer has a unique name in a Scene, and it should use one tileset. The data is a vector. -1 represents empty. XY represents Row X and Column Y, 0<=X,Y<=9999. XY / 1000 -> X, XY % 1000 -> Y
